Instructions on How To Create a Personal Web Page on GitHub

1. First, create an account on GitHub. When you make your username, make sure it's something that can be easily typed and remembered, should other people need to come find it.\n 2. On the home page of GitHub, you will see a button with the text 'Create New Repository". Name it something that can be easily typed. 3. Once you do that, you'll see an option to upload an existing file or create a new one. Choose "Upload Existing File". 4. Go to VS Code and create a file named index.html. Put in the following code. '''''

[Your Name] Portfolio

''''' 5. Drag and drop both files (the index.html and the file of the image) into GitHub. Make sure they are under the same folders in your Finder (for Mac) or equivalent to Finder for other devices and that the file path is correct if the image doesn't show up. 6. Make sure to click 'Commit Changes'. 7. Open a new tab in your browser and go to 'yourusername'.github.io/'the name of your current repository'. 8. If the image doesn't load, got to settings (at the top menu bar in GitHub), go under 'Code and automation', then 'Pages'. 9. Make sure under the heading "Branches", that the selected branch is 'main'. Refresh and under the same place in Settings, you should see a little box that says 'Visit Site'. NOTICE: it can take up to 10 minutes for your site to publish. Hope this helped! :)
